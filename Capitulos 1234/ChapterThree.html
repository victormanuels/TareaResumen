Tipos de Variables

Al igual que la mayora de las otras plataformas de anlisis de datos, R admite un conjunto de tipos de variables predefinidos, incluidos los numricos, de caracteres, lgicos y factores. ### Variables numricas vs.ordinales vs.nominales Una caracterstica importante de los datos numricos es que podemos aplicarle muchas operaciones matemticas, calcular sumas, diferencias, productos, cocientes, promedios, races cuadradas y muchas otras combinaciones y / o transformaciones. Sin embargo, no todas las variables son numricas, y la mayora de estas operaciones matemticas no son aplicables a las variables no numricas. Al trabajar con variables nominales, todo lo que podemos hacer es contar y comparar, haciendo preguntas como las siguientes, dada una variable categrica:

Cuntos valores distintos o "niveles" exhibe la variable?
 Con qu frecuencia se produce cada uno de estos niveles en el conjunto de datos?
Cmo vara el comportamiento de otra variable X sobre los niveles de C?

Estas preguntas forman la base para las caracterizaciones de variables categricas Otra clase de variables que es en muchos aspectos intermedia entre las variables numricas y nominales es la clase de variables ordinales, tambin denominadas variables categricas ordenadas o factores ordenados. Estas variables asumen valores no numricos, por lo que el rango completo de operaciones matemticas no est disponible para nosotros al trabajar con ellas, pero poseen un orden inherente, por lo que podemos decir que un valor de la variable es menor que o precedeOtro valor. Como ejemplo especfico, la variable Make en el framework Cars93 del paquete MASS se incluye como una variable de factor, pero se puede convertir a un vector de caracteres de longitud 93, a partir del cual la mediana se puede calcular como:

El trmino inliers a veces se usa para referirse a los puntos nominales o no perifricos en una secuencia de datos. Un inlier es un valor de datos que se encuentra en el interior de una distribucin estadstica y est en error. Una caracterstica comn de las inliers en los datos numricos es que a menudo representan el mismo valor, que se repite con una frecuencia inusual.
